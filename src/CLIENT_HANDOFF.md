### Common Issues

**"Application submission failed"**
- Solution: Check OpenAI API key is configured
- Check: API credits available
- Action: Refresh and try again

**"AI screening stuck"**
- Solution: Check OpenAI account has credits
- Action: Wait 60 seconds and refresh
- Backup: Manual screening still possible

- Solution: Ensure account was created first
- Check: Password is 6+ characters
- Action: Use "Forgot password" (if enabled)

- Solution: File must be PDF
- Check: File size under 10MB
- Action: Compress PDF or use different file
